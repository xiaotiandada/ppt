vuejs简介

Vue 是一套用于构建用户界面的渐进式框架。

vuejs开发环境搭建

  1. 安装nodejs

  2. 安装vue cli

  3. 初始化vue项目并启动

安装nodejs

下载地址: https://nodejs.org/zh-cn/download/

  1. 下载对应的 nodejs LTS 安装包

  2. 检查是否安装成功

$ node -v
v10.6.0

$ npm -v
6.1.0

安装vue cli

  • Install:
npm install -g @vue/cli    # 安装 vue-cli 工具
# OR
yarn global add @vue/cli
  • Create a project:
vue create my-project    # 创建vue项目
# OR
vue ui

  • 这里是最新版vue-cli3

  • 目前开发可以采用以前的方式

vue init webpack my-project  # 初始化vue项目

这样会集成 webpack 工具,并且工具已经事先配置好了

初始化vue项目并启动

安装完成之后根据提示操作

cd my-project # 进入目录

npm install   # 安装依赖

npm run dev   # 启动服务

localhost:8081 # 默认端口
